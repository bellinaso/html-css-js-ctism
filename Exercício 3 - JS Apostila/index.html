Exercício 7

Window, document e select

window: Objeto que representa a janela do navegador. Contém propriedades e métodos relacionados à janela.

document: Objeto que representa o documento HTML carregado na janela. Oferece métodos para interagir com elementos HTML.

select: Uma tag HTML usada para criar listas suspensas (dropdowns) em formulários. Permite que os usuários escolham uma opção entre várias.
option: Cria opções dentro do select.
value: Define o valor associado a cada opção.
Texto entre as tags option: É o texto visível para o usuário na lista suspensa.

Exercício 8

Input

input: Uma tag HTML utilizada para criar campos interativos em formulários. Pode ser de vários tipos, como texto, número, checkbox, radio, entre outros.
type: Define o tipo de input (texto, número, checkbox, etc.).
placeholder: Texto exibido como dica dentro do campo.
min e max: Limites para um input numérico.
value: Valor padrão para um input.
